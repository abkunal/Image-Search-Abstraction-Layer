Image Search Abstraction Layer



Searches the image by the given query on the web and returns a JSON file with image's url, snippet, thumbnail and context.

In place of "query" enter your search term. Limit number of results using offset (1-10).

Example Usage:

To get the search results

https://kunal-image-search-abstraction.glitch.me/api/imagesearch/query?offset=10
https://kunal-image-search-abstraction.glitch.me/api/imagesearch/freeCodeCamp?offset=10

To get the last 10 search queries type:

https://kunal-image-search-abstraction.glitch.me/api/latest/imagesearch/
